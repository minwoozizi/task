# 도매 영업관리 포털

통신사(KT) 도매채널 영업관리 시스템. 7본부 × 42센터의 실적을 실시간으로 관리합니다.

## 기능
- 장려금투명화 업로드
- 시장동향보고 (모델별 단가, 추가정책, 3사 특이동향)
- 유무선 미개동수량
- 보증보험증권 관리
- 접점담당M 확인
- 실시간 실적현황

## 권한
- **영업사원**: 자기 센터 데이터 입력 (바로 진입)
- **관리자**: 전체 조회 + 삭제 + 초기화 (비밀번호 `1234`)

## 데이터 저장
Supabase 원격 DB로 팀 전체가 실시간 공유합니다.

## 배포
Netlify Drop 또는 GitHub 연동으로 바로 배포 가능.
